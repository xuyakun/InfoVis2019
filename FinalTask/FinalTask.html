Overview:

The page is divided into three parts according to Description(30%):Rendering screen(50%):UI(20%). The Rendering screen is used to show the result of visualization and the UI is used to change the visualization parameters.

Introduction:

From this UI, the following functions can be implemented:

1.Change isovalue.
We can change isovalue from 0 to 255.

2.Change color.
We can change color of lobster by changing the value of R,G,B from 0 to 255.

3.Shader.
We can implement shaders based on different reflection models to change the meterial of lobster by four buttons. 1.Normal: Change the meterial to MeshLambertMaterial; 2.Lambertian: Implement Gouraud shading based on Lambertian reflection; 3.Phong: Implement Gouraud shading based on Phong reflection; 4.BlinnPhong: Implement Phong shading based on Blinn-Phong reflection.

4.Reset.
We can reset parameters to initial value.
Initial value:
Isovalue: 128
Color: R: 150; G: 80; B: 100.
Meterial of lobster: MeshLambertMaterial
